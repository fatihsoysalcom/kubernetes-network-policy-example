# Kubernetes Network Policy Example

This example demonstrates a basic Kubernetes NetworkPolicy. It shows how to restrict network traffic between pods, allowing only specific communication paths. This is a core concept for securing microservices in a Kubernetes environment, similar to what Tigera Lynx provides at an advanced level.

## Language

`yaml`

## How to Run

Apply this YAML to your Kubernetes cluster using 'kubectl apply -f network-policy.yaml'.
